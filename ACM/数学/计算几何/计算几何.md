##### 点乘与叉乘

(x1,y1)×(x2,y2)=x1*y2-x2\*y1;

(x1,y1)·(x2,y2)=x1*x2+y1\*y2;

```c++
inline double cross(P a,P b){ //外积
	return a.x*b.y-a.y*b.x;
}
```

```c++
inline double dis(P a,P b){  //求距离
	return sqrt((a.x-b.x)*(a.x-b.x)+(a.y-b.y)*(a.y-b.y));
}
```
